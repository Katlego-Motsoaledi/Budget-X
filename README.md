Budget X is a straightforward and effective expense tracking app. It helps users keep an eye on their spending, manage daily expenses, and evaluate their financial habits over time. 

Features  
User authentication (Login and Registration)  
Monthly expense summary  
Add new expenses with detailed inputs  
Upload expense photos (for example, receipts)  
View full expense history  
Filter expenses by date  

User Flow  
1. Launching the App  
When users open Budget X, they see the Home/Login page. They can log in if they have an account or register as new users.  
2. Authentication  
Login  
Returning users enter their credentials to access their accounts.  
Registration  
New users click "Register here," fill in their personal information, and submit to create an account.  
3. Dashboard  
After logging in, users go to the dashboard where they can:  
View total expenses for the current month  
Access these options:  
Add New Expenses  
Expenses History  
4. Adding New Expenses  
Click the "Add New Expenses" button to record a new expense.  
Required fields:  
Amount spent  
Date of expense  
Start time  
End time  
Description  
Category  
Optional:  
Upload a photo (for example, a receipt)  
Action: Click "Save Expenses" to store the entry.  
5. Viewing Expense History  
Click the "Expenses History" button to see all recorded expenses.  
Filtering:  
Click the filter button  
Select a date range  
View expenses within the selected period  

Usage Summary  
Login or register  
View monthly spending  
Add new expenses  
Review and filter past expenses  

Notes  
Each expense can include a photo for better record-keeping.  
Filtering helps track spending over specific periods.  
The app is designed for simplicity and ease of use.
